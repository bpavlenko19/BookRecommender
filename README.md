Звіт по дослідницькому проекту

Тема: Рекомендаційна система книг для онлайн-бібліотеки
Автор: [Павленко Богдан]
Курс: Інтелектуальний аналіз даних

1. Мета та завдання проекту
Мета: Розробка системи, яка аналізує поведінку користувачів і пропонує книги на основі їх інтересів та оцінок.
Основні завдання:
Підготовка та очищення даних.
Проведення попереднього аналізу даних (EDA).
Створення матриці користувач–книга для рекомендацій.
Розробка алгоритму рекомендацій.
Оцінка ефективності системи.
Візуалізація результатів.
