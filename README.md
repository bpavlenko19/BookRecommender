Опис проєкту

Репозиторій book-recommender — навчально-дослідницький проєкт з інтелектуального аналізу даних (Data Mining).
Основна мета — створення системи персоналізованих рекомендацій книг для онлайн-бібліотеки на основі інтересів користувачів та їхньої поведінки.

Проєкт моделює взаємодію користувачів з книгами, аналізує оцінки, історію переглядів і популярність книг, а також пропонує персональні рекомендації.

Особливості даних

Структура вхідних файлів:
Поле	Опис
ISBN	Унікальний ідентифікатор книги
Book-Title	Назва книги
Book-Author	Автор книги
Year-Of-Publication	Рік видання
Publisher	Видавництво
User-ID	Ідентифікатор користувача
Book-Rating	Рейтинг книги, який поставив користувач
Location	Локація користувача
Age	Вік користувача
Файли CSV (books.csv, ratings.csv, users.csv) використовуються як вхідні дані для більшості скриптів у репозиторії.
