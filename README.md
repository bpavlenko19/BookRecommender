Опис проєкту
Репозиторій book-recommender навчально дослідницький проєкт з інтелектуального аналізу даних (Data Mining).
Проєкт присвячений розробці системи персоналізованих рекомендацій книг для онлайн-бібліотеки. Система аналізує поведінку користувачів, їхні оцінки, історію переглядів та взаємодії з книгами, щоб пропонувати найбільш релевантні та цікаві для кожного користувача книги.
Основна мета проєкту - створення ефективного інструменту персоналізованих рекомендацій, який враховує:
історію оцінок користувачів;
популярність та рейтинги книг;
жанрові та авторські вподобання;
поведінку та активність користувачів на платформі.
Проєкт моделює процеси взаємодії користувачів з бібліотечною платформою та дозволяє автоматизувати підбір книг, підвищуючи залучення користувачів і покращуючи досвід взаємодії з онлайн-бібліотекою.
